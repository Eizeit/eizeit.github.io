    树的性质

    1. 树中的结点数等于所有结点度数之和加$1$
    2. 度为$m$的树中第$i$层上最多有$m^{i-1}$个结点（$i \ge 1$）
    3. 高度为$h$的$m$叉树最多有$\frac{m^h - 1}{m-1}$个结点（等比数列求和）
    4. 具有$n$个结点的$m$叉树的最小高度为$\left \lceil log_m(n(m-1)+1) \right \rceil$（用上一条反解一下）
