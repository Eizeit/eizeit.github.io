【考研英语】 2012年考研英语一词组、单词

Sun 2020-10-04
  • 阅读理解

2012年考研英语真题里个人记忆模糊的单词、词组。

完形填空

  • restore [v.] 恢复；修复
  • be bound by 受…约束
  • be subject to 使服从，受…管制
  • ethics code 道德规范（code：法律；法规）
  • framer [n.] 制定者；创作者；成帧器
  • envision [v.] 展望；想象
  • cultivate [v.] 耕作；种植；栽培；培育；建立（友谊）；结交（朋友）；获得（支持）
  • dismiss [v.] 不予考虑；去除；消除；解雇；开除
  • the code of conduct 行为准则

  • accountable [adj.] （对自己的行为、决定）负有责任

    • be accountable to sb. 对某人负责
    • be accountable to the laws/the code of conduct 遵守法律/行为准则
    • be accountable for sth. 对（做过的）某事负责

阅读理解

Text1

  • supplement

    • [n.] 增补(物)；补充(物)；添加物；(报纸的)增刊；(书籍的)补编，补遗，附录
    • [v.] 增补；补充

  • undesirable

    • [adj.] 不想要的；不得人心的；易惹麻烦的；不良的
    • [n.] 不受欢迎的人；不良分子

Text2

  • renege on 背弃；违背
  • condemn 谴责
  • reaffirm 重申

  • dishonor

    • [v.] 违背；使蒙羞
    • [n.] 不名誉；拒付；丢脸

  • secure

    • [v.] 获得；实现；栓牢；保护
    • [adj.] 安心的；有把握的；可靠的；稳固的

Text3

  • inspection [n.] 视察；检查；查看；审视

Text4

  • prudent [adj.] 谨慎的
  • clamp down 取缔；压制；管制；镇压
